> End-to-end medical AI system for automated blood cell classification — from raw image to real-time diagnosis in under 50ms.

| Metric | Value |
|--------|-------|
| 📊 Test Accuracy | **95%+** |
| ⚡ Inference Speed | **< 50ms** |
| 🔬 Cell Classes | **8** |
| 🔒 Data Retained | **Zero** |

**Production-grade features:**

- ✅ Drag-and-drop web UI with real-time confidence scores
- ✅ REST API — secure in-memory processing, nothing persisted
- ✅ Reproducible training pipeline — seed control, versioned datasets, config files
- ✅ ONNX / TorchScript export-ready for edge & embedded deployment
- ✅ Dockerized — one-click Railway deploy template
